# Region Tech

Official website and digital infrastructure repository for Region Tech.

## Overview

This repository contains the source code, assets, and deployment configuration for the Region Tech website.

The project is being built as a modern static website hosted through Cloudflare Pages and managed through GitHub.

## Goals

- Build a scalable digital presence for Region Tech
- Create a clean and modern responsive website
- Develop service pages, project showcases, and contact systems
- Integrate future AI and automation tools
- Maintain fast, secure, and lightweight hosting infrastructure

## Tech Stack

- HTML
- CSS
- JavaScript
- GitHub
- Cloudflare Pages
